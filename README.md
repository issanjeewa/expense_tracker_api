
## Description

Sample expence tracker api backend using [Nest](https://github.com/nestjs/nest) framework.
<br/> <br/>
includes:
- Passport JWT Authentication
- Swagger API Documentation

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
